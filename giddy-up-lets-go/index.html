Giddy Up Let's Go

01 Mar 2018 | Brian Hooper


Whether is it a full featured API, rich CLI, simple Library or Lambda Function Golang has quickly become my go to language (no pun intended) for everyday coding. Go is an open source programming language that makes it easy to build simple, reliable, and efficient software.

It is a relatively simple language and was designed with a very minimalistic approach. Go version 1 guarantees compatibility for the language specification and major parts of the standard library. All versions up to the current Go 1.12 release have maintained this promise.

Features

Here are just a few of the core features:

1. Build and Run

Go generates binaries for your applications with all the dependencies built-in that can run on any OS or Architecture.

2. Static Typing

Go is a statically typed language.

3. Standard Library

Go comes with a powerful standard library, distributed as packages.

4. Concurrency

Go offers some great concurrency primitives and makes it extremely easy to implement a concurrent system.

5. Testing Support

Go Language brings Unit Testing right into the language itself.
