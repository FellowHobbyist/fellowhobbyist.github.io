My Toolbox

01 Jan 2018 | Brian Hooper


This post is a currated list of Frameworks, Languages, Libraries, and Tools.

There is no single tool which fits all needs. There are many fantastic open source tools available for developing solutions that cover applications, infrastructure, and databases. In addition there is quite a bit of tooling which come under various categories like infrastructure as code, configuration management, orchestration, continuous integration, monitoring, networking, security, and etc.
