A Quick Guide to Getting Started with Hugo

February 01, 2017

Summary This post is a simple guide to quickly get up-and-running with Hugo and GitHub Pages for hosting your next static website over HTTPS via a Custom Domain. We’ll also be working with Git Submodules. As it often happens while working on one project, you will need to use another project from within it. We will be using Git Submodules to accomplish the following: Manage the Source Code via a Private Git Repository Publish the Static Site to a separate GitHub Pages Repository Source a Custom Hugo Theme You can learn More About Git Submodules Here
